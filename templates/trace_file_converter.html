Understanding SSD Trace Files and Details of Each Field
Field Name Example Description
Device Major Number 259 The major number of the device generating the trace.
Device Minor Number 2 The minor number of the device generating the trace.
CPU Core ID 0 The ID of the CPU core involved in the operation.
Record ID 1 The unique identifier for the trace record.
Timestamp (in nanoseconds) 0.016192694 The timestamp indicating when the trace event occurred, measured in nanoseconds.
ProcessID 4020 The ID of the process associated with the trace event.
Trace Action Q Queueing: The operation is being queued for execution.
G Get: The operation is a read from the device.
U Unqueueing: The operation is being removed from the queue without being executed.
I Insert: The operation is being inserted into the queue.
D Delete: The operation is being deleted from the queue.
C Complete: The operation has completed execution.
OperationType R Read: The operation is a read from the device.
N None: No specific operation type.
RS Read Synchronous: The read operation is synchronous.
W Write: The operation is a write to the device.
SectorNumber + I/O Size 991034800 + 256 The sector number and size of the I/O operation, typically in bytes.
ProcessName [java] The name of the process associated with the trace event.
